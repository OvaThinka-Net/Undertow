Step 1: Create a Google Cloud Project

Undertow uses Google Drive as its transport layer. You need a free Google Cloud project to get API access.

  1. Go to Google Cloud Console — New Project
  2. Enter a project name (e.g. undertow) and click Create
  3. Wait a few seconds for the project to be created
  4. Make sure the new project is selected in the top dropdown bar
💡 This is completely free. Google Cloud projects don't cost anything. The Drive API has generous free quotas.

Step 2: Enable the Google Drive API

Enable the Drive API so Undertow can store and retrieve data:

  1. Go to Google Drive API page
  2. Make sure your project is selected in the top bar
  3. Click the blue Enable button
  4. Wait until the page shows "API enabled"
💡 If you see "API enabled" already, it's already turned on — just proceed.

Step 3: Create OAuth Credentials

A) Configure Google Auth Platform

  1. Go to Google Auth Platform
  2. If you see "not configured yet", click Get Started
  3. App Information: Enter an app name (e.g. Undertow) and your email → click Next
  4. Audience: Select External → click Next
  5. Contact Information: Enter your email → click Next
  6. Click Finish
🚨 Required: The app starts in "Testing" mode which blocks sign-in. Go to Audience in the left sidebar and click Publish App → Confirm.
Without this, you'll get "Access blocked" (Error 403) when authorizing.
Publishing also prevents your token from expiring every 7 days.

B) Create an OAuth Client ID

⚠️ You must complete part A first — the Clients page won't work until the Auth Platform is configured.

  1. In the left sidebar, click Clients
  2. Click + Create Client
  3. Set Application type to Desktop app
  4. Name it anything (e.g. Undertow)
  5. Click Create
  6. Click the ⬇ Download button (download icon) to save the JSON file
✅ You should now have a client_secret_*.json file downloaded to your computer.

B) Share with the client user

The client needs access to this folder. Share it with the Google account that will run the client:

  1. Open Google Drive
  2. Find the folder you just created (shown above)
  3. Right-click → Share
  4. Enter the client user's Gmail address and set role to Editor
  5. Click Send
💡 If server and client use the same Google account, you can skip this — both sides already see the folder.
